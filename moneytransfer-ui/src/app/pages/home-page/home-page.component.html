Getting Started

Installing and Deployment

The Project is composed by one parent pom that import two modules:

  • moneytransfer-server
  • moneytransfer-web

the first contain the backend code (REST API) and the second contain the client side application (that use API exposed from server)

Project requires Maven v. 3.5.0 to run.

On root folder run:

$ mvn clean install

this command will generate the moneytransfer-app.war file.

Move to moneytransfer-server directory and run:

$ mvn spring-boot:run

Now you use the application on http://localhost:8080/

Database access:

The application provided is configured to automatically use the in-memory database H2 and to automatically create the tables at startup time for every domain entity modelled.

Whenever you restart the application, the database will be reset. To access its database administration panel, open the following url in the browser after having started the application:

  • http://localhost:8080/h2/

And use for the databaseURL: jdbc:h2:mem:bank
